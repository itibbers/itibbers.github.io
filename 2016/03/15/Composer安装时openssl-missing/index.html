Composer安装时openssl missing

Mar 15, 2016 php php composer error

问题描述：“The openssl extension is missing, which means that secure HTTPS transfers are impossible.”

确保php.ini中extension=php_openssl.dll打开。

php.ini文件在apache和php目录下都有，建议都打开。
