一句话解压目录下所有gz文件

Jul 26, 2017 linux linux gz shell

服务器迁移过程中，把www目录下所有文件通过 tar -zxvf 命令打包，再下载上传到新服务器，再解压。
但是解压后发现个问题，部分文件带着.gz文件后缀。（压缩后有15G大小，难道解压的不充分？）

如果是.gz压缩包(不带tar)，用gzip命令即可(-d选项可以解压)，格式是
gzip 压缩包名(包括后面的.gz) -d 解压位置
如果不指定解压位置，将解压到当前工作文件夹里
